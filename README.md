# Professional-Lawyer

Next.js + Tailwind CSS law firm website scaffold.

## Structure

- `src/app` - App Router pages, layout, global styles, API routes
- `src/components` - Shared UI components
- `src/sections` - Homepage sections
- `src/lib` - Site content and data
- `.env.example` - Environment variable template

## Run

```bash
npm install
npm run dev
```
